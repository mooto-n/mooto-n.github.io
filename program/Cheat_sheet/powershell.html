定義

$variable = "" # 型は不要(定義も可能) クラス内は型が必須
  • 数値: [integer]
  • 文字列: [string]
  • 辞書型: [hashtable]
  • クラス定義: class Class_Variable{}
  • コンストラクタ: class名と同じ名前のメソッドを作成する(型は不要)

コマンド

  • ファイルコピー: Copy-Item <コピー元> <コピー先>
  • ファイル移動: Move-Item <移動元> <移動先>
  • 文字列検索: Select-String

文字列操作

  • 文字列入力: $Variable = Read-Host "<文字列>"
